# AI Voice Visualizer

A futuristic, real-time voice visualizer built with React, TypeScript, Canvas, and the Web Audio API.

## Features

- Live microphone visualization with FFT frequency analysis
- Cyber portal waves that grow with voice volume
- Browser-based live transcription and text-to-speech playback
- Audio recording, upload, playback, and download
- Smooth responsive canvas animation for desktop and mobile

## Run locally

```bash
npm install
npm run dev
```

Open the local Vite address shown in the terminal, then allow microphone access to start visualizing your voice.
